# Schedu


## Project

- The main porpuse of this project is to build a web based application that could be used by clients and businesses. A web platform where businesses could display the available services and information allowing the users to schedule those services.
Users could have access to the business agenda and, without spending time calling or texting, schedule the wanted service for an available time slot.

- One of the stronges porpuses of this project is also to make a efficient demostration of Software Development skills, turning the Schedu repository public and available to everyone. The owner belief is practicing to archieve perfection. 


## Technologies 

- This project was build based on ASP.NET Core MVC architecture. 
- .NET 6 and EntityFramework Core 6

## Database

- In order to run the application locally connected to the database it is required to have a running local SQL instance. 
- For this project the database in use is (localdb) which is the default local database created by SQL Server and the instance name is Local. 
	- To perform the Code First Migration and create the local database to successfuly be used by Schedu it is only required to execute the command "Update-Database" in the Package Manager Console in Visual Studio
